ZimaOS V1.3.0

Download

For users who have installed ZimaOS V1.1:
Click on the red dot in the top left corner of the dashboard to initiate the update.

Installer: https://github.com/IceWhaleTech/ZimaOS/releases/download/1.3.0-2/zimaos_zimacube-1.3.0-2_installer.img

Manual upgrade: https://github.com/IceWhaleTech/ZimaOS/releases/download/1.3.0-2/zimaos_zimacube-1.3.0-2.raucb

Brand New

The settings panel has been completely redesigned, and disk health is displayed

ZVM virtual machines support one-click installation of Windows/Linux

New

Video files in Files now support preview

PCIE disks not in bays support RAID

ZVM virtual machines are compatible with Windows 11 installation

ZVM virtual machines support network settings to None

Files now support previews in .doc, .xls, .ppt, and video thumbnails

Supports identification and display of wireless network cards

Fixed

Fixed the occupation of ports 1900 and 1901 to ensure Plex is available

Improved the priority of Thunderbolt connection to ensure high-speed transmission

Automatically calibrate the system time to the time zone

Compatible with iPhone connection to Samba

Increased the stability of application data migration

Files fixed the problem that the cloud disk could not be unlink

Improved some interface issues in the use of ZVM virtual machines

Fixed a lot of interface display and operation issues

Storage disappears updated on some machines

Disk hibernation policy failure after reboot issue

Storage disappears updated on some ZimaCube Pro machines
