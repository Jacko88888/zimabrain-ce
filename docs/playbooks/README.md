# ZimaBrain CE Playbooks

v1.5.0 handles verifier-first diagnostics.

v1.6.0 adds official-doc-backed how-to playbooks for common ZimaOS user questions.

Playbook answer rule:

1. If same-report evidence exists, use diagnostic layer first.
2. If the question is a how-to, use official ZimaOS docs playbook.
3. If it is a forum title with no evidence, ask for the safest missing evidence.
4. Never pretend a how-to answer is verified from the user's current report.
